# Aussie Agent Skills

The Australian AI agent skills marketplace. Open-source skills for Claude Code, Cursor, GitHub Copilot, and more.

## Structure

- `/skills` — Australian-specific AI agent skills
- `/website` — Next.js marketplace website
- `/cli` — Command-line installer tool

## Domains

- aussieagentskills.com
- aussieagentskills.com.au
- agentskills.com.au

## Categories

- Tax & Finance (ATO, BAS, Super)
- Government Services (myGov, ABN, Medicare)
- Legal & Compliance (Fair Work, Privacy Act)
- Business Operations (Australian standards)
- Localization (Australian English, slang)
