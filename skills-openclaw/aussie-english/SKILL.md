# Australian English Localisation

Ensure all outputs use correct Australian spelling, terminology, and cultural context.

## When to Use

- All outputs for Australian audiences
- User explicitly wants Australian English
- Content involves Australian business, government, or culture

## Core Rules

### 1. British-Based Spelling
colour, favour, organise, centre, metre

### 2. Australian Terminology
Super not 401(k), TFN not SSN, postcode not ZIP

### 3. Metric System
Kilometres, kilograms, Celsius. Always.

### 4. Date Format
DD/MM/YYYY (24/03/2025)

### 5. Casual but Professional
"No worries" is acceptable. Avoid American hype.
