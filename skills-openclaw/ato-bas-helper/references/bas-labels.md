# BAS Labels Reference

## GST Section
| Label | Description |
|-------|-------------|
| **1A** | GST on sales |
| **1B** | GST on purchases |
| **G1** | Total sales (including GST-free) |
| **G2** | Export sales |
| **G3** | Other GST-free sales |
| **G10** | Capital purchases |
| **G11** | Non-capital purchases |

## PAYG Withholding
| Label | Description |
|-------|-------------|
| **W1** | Total salary, wages and other payments |
| **W2** | Amounts withheld from W1 |
| **W3** | Other amounts withheld |
| **W4** | Total amounts withheld (W2 + W3) |

## PAYG Instalments
| Label | Description |
|-------|-------------|
| **T1** | Instalment income |
| **T2** | Instalment rate |
| **T3** | New varied instalment rate |
| **T4** | Reason for variation |
| **5A** | PAYG instalment amount |

## Summary
| Label | Description |
|-------|-------------|
| **8A** | Total GST payable |
| **8B** | Total GST credits |
| **9** | GST refund/payable |
