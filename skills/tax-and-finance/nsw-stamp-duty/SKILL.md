# NSW Stamp Duty (Transfer Duty)

You are an expert in NSW transfer duty (stamp duty) for property transactions.

## Standard Duty Rates (2024-25)

| Property Value | Duty Rate |
|----------------|-----------|
| $0 - $17,000 | 1.25% |
| $17,001 - $36,000 | $212 + 1.5% over $17,000 |
| $36,001 - $97,000 | $497 + 1.75% over $36,000 |
| $97,001 - $364,000 | $1,565 + 3.5% over $97,000 |
| $364,001 - $1,212,000 | $10,910 + 4.5% over $364,000 |
| $1,212,001 - $3,636,000 | $49,070 + 5.5% over $1,212,000 |
| Over $3,636,000 | $182,390 + 7% over $3,636,000 |

**Premium property rate:** 7% applies to residential properties over $3,636,000

## First Home Buyer Benefits

### Full Exemption
- New homes: Up to $800,000
- Existing homes: Up to $650,000 (changed from July 2024)
- Must be a new loan, not refinancing

### Reduced Duty
- New homes: $800,001 - $1,000,000 (sliding scale)
- Existing homes: $650,001 - $800,000 (sliding scale)

### Eligibility Requirements
1. Natural person (not company/trust)
2. Over 18 years old
3. Australian citizen or permanent resident
4. Never owned property in Australia
5. Move in within 12 months
6. Live there for at least 6 continuous months

## First Home Owner Grant (FHOG)

- **Amount:** $10,000
- **New homes only:** Under $600,000
- **Can combine with duty exemption**

## Foreign Buyer Surcharge

- **Additional 8%** for foreign purchasers
- Applies on top of standard duty
- Some exemptions for visa holders

## Quick Calculations

### Example 1: First home $600,000 (existing)
- Standard duty would be: ~$22,490
- **First home buyer: $0** (exempt under $650,000)

### Example 2: Investment property $800,000
- $10,910 + ($800,000 - $364,000) × 4.5%
- $10,910 + $19,620 = **$30,530**

### Example 3: Premium property $4,000,000
- $182,390 + ($4,000,000 - $3,636,000) × 7%
- $182,390 + $25,480 = **$207,870**

## Off-the-Plan Concessions

- Duty calculated on land value only (not building)
- Building value can be deducted if contract signed before completion
- Maximum 12-month deferral available

## When Duty is Due

- Within **3 months** of settlement
- Late payment incurs interest
- Lodge via Revenue NSW portal

## Common Mistakes

1. Forgetting foreign surcharge
2. Missing first home buyer residency requirements
3. Not claiming off-the-plan concession
4. Buying through company/trust (no first home benefits)
